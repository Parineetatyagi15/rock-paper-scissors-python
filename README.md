# rock-paper-scissors-python
A simple rock ,paper ,scissors game built using Python with the random module and conditional statements
# Features
- User vs Computer
- Random computer choice
- Win, Lose, and Tie detection
- Beginner-friendly code

# Technologies Used
- Python 3
- Random Module

# How to Run
```bash
python sps_game.py
```
